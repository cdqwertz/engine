#Shortcuts

###Code Editor

- S : toggle snap
- Q : increase snap distance
- A : decrease snap distance

###Prefab Editor

- S : toggle snap
- Q : increase snap distance
- A : decrease snap distance
- W : add actor
- E : add component
- T : add transform component

###Level Editor

- S : toggle snap
- Q : increase snap distance
- A : decrease snap distance

###Event Editor

- S : toggle snap
- Q : increase snap distance
- A : decrease snap distance
